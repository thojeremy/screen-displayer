# Screen displayer

A simple project made using Electron

## Development

### Dependencies

In order to install all dependencies, run :

```sh
npm i
```

### Launch unit tests

Tests will have to be done when dev will start.

In order to run unit tests, run :

```sh
npm run test
```

### Launch application

Launch the following command :

```sh
electron .
```
